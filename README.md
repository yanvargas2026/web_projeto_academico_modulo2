# Coleta Lixo --- Protótipo Plataforma Web

## Descrição do Projeto

O **Plataforma coleta de lixo eletroeletronico** é uma aplicação web dinâmica desenvolvida como parte
do **Projeto Integrador II**, com o objetivo de facilitar o registro e
acompanhamento de solicitações de coleta de lixo eletrônico na cidade de
Corumbá-MS.\
A aplicação permite que o usuário registre uma solicitação, visualize o
status de coleta e solicite o cancelamento conforme regras definidas
pela Secretaria de Infraestrutura.

------------------------------------------------------------------------

## Tecnologias Utilizadas

-   **HTML5** --- Estrutura semântica e acessível da aplicação.\
-   **CSS3** --- Estilização responsiva para desktop e mobile.\
-   **Vue.js (CDN)** --- Framework JavaScript para interface dinâmica.\
-   **LocalStorage** --- Armazenamento local dos dados do usuário.\
-   **Git & GitHub** --- Controle de versão e publicação do código.

## Funcionalidades Principais

-   Registro de solicitações de coleta (nome, endereço e tipo de lixo
    eletrônico).\
-   Armazenamento automático das solicitações no navegador.\
-   Controle de status da solicitação:
    -   **Aguardando**
    -   **Recebida**
    -   **Cancelado**
    -   **Cancelamento Solicitado**
-   Cancelamento inteligente com validação do status atual.\
-   Interface moderna e responsiva.
